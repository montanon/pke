# Security

PKE is a prototype for verifiable chain of custody for encrypted mobile evidence. It is not a production trust-and-safety system.

## Reporting security issues

Report security issues privately using [GitHub's private vulnerability reporting](https://github.com/montanon/pke/security/advisories/new). Do not submit sensitive reports in public GitHub issues, discussions, or pull requests.

A report may include:

- affected component,
- high-level impact,
- reproduction steps using synthetic data,
- expected behavior,
- actual behavior,
- suggested mitigation.

## Do not submit publicly

- illegal content or suspected illegal content,
- real evidence media,
- personal information,
- private keys or credentials,
- exploit proof-of-concepts against live systems,
- screenshots containing sensitive data.

## Disclosure

Please avoid public disclosure of exploitable vulnerabilities before maintainers have had a reasonable opportunity to investigate and mitigate.

For full details, see [context/14_security_reporting.md](./context/14_security_reporting.md).
